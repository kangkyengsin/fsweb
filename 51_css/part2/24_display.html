블록레벨 요소

  • display:block;(단, li는 display:list-item;)
  • 줄바꿈 : 위에서 아래로 배치(세로 배치)
  • 기본값이 width:100%(width:auto); height:콘텐츠크기(height:auto);border:none;
  • width, height, margin, padding, borderfmf를 지정할 수 있다.
  • div, h1~h6, p, ul, li, ...

인라인 요소

  • display:inline;
  • 줄바꿈 : 왼쪽에서 오른쪽으로 배치(가로 배치)
  • span, img, a, em, b, i, label, ...
  • width:콘텐츠크기; height:콘텐츠크기; border:none;
  • width와 height는 지정되지 않는다. (단, 이미지는 예외)
  • margin은 좌우만 적용된다.
  • padding, border는 모두 적용된다.

인라인 요소

  • display:inline-block;
  • 줄바꿈 : 왼쪽에서 오른쪽으로 배치(가로 배치)
  • button, input, ...
  • width, height, margin, padding, border 모두 지정된다.
  • button은 box-sizing:border-box; 이다.
